# Introduction

Why keep reinventing the wheel? No website needs to be built from scratch anymore because with WP_Lemon you can rapidly create beautiful websites by using our in-house modules or create ones yourselve easily. Our modules consist of templates, navigation menus, content blocks, sliders, carousels, forms, frequently asked questions, blog and social media functions and more ...

And we are just getting started! Take a look at our [Changelog!](/changelog) for the latest update.
