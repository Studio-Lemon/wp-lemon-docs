# wp-lemon Documentation

Welcome to the **wp-lemon** documentation! wp-lemon is your go-to WordPress theme for creating good looking and lean websites.

## What is wp-lemon?

wp-lemon is a modern WordPress theme framework that combines the power of:
- **Timber** - For templating with Twig
- **ACF (Advanced Custom Fields)** - For flexible content management
- **Bootstrap** - For responsive design
- **Bedrock** - For better WordPress project structure
- **Bulldozer** - For asset management

## Getting Started

To get started with wp-lemon, follow these steps:

1. Check the [Requirements](/requirements) to ensure your environment is ready
2. Follow the [Setup Guide](/setup) to create a new project
3. Read the [Getting Started](/getting-started) guide to begin development

## Key Features

- **Block-based Development** - Create custom Gutenberg blocks with ease
- **MVC Architecture** - Organize your code with models, controllers, and views
- **Customizer Integration** - Configure your site through WordPress customizer
- **Performance Optimized** - Built with performance in mind
- **Developer Friendly** - Clear structure and extensive documentation
